- **降低显存方法**:
  1. 降分辨率 (832×480 → 624×624)
  2. 降帧数 (81帧 → 49帧)
  3. 用 fp8 量化 T5 (`quantization: "fp8"`)
  4. offload T5 到 CPU (`load_device: "offload_device"`)
- **人物扭曲原因**:
