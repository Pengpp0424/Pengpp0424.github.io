## 09:33 Pre-compaction flush — Win11 升级命令问题

### 问题命令
```
reg add HKLM\SYSTEM\Setup\MoSetup _DWORD /d 1 /f
```

### 问题原因
语法错误：`HKLM\SYSTEM\Setup\MoSetup` 后面缺少注册表**值名称**（value name）。`reg add` 需要完整路径，含子项+值名，格式应为：
```
reg add HKLM\SYSTEM\Setup\MoSetup\ValueName /t REG_DWORD /d 1 /f
```

不过此法绕过 Win11 硬件限制已被微软封堵，不推荐。

### 推荐升级方法
| 方法 | 说明 |
|------|------|
| PC 健康检查 | 微软官网下载，检测硬件兼容性 |
| Win11 安装助手 | 微软官网下载，原地升级 |
| Media Creation Tool | 制作U盘安装，最稳妥 |
