## I2V vs T2V 路线讨论（2026-05-16 20:54）

### 老鹏的疑问

   - I2V 用参考图锚定主角外观，每帧基于同一张图，一致性有保障

2. **LoRA 在 I2V 同样可用**
   - 我们的工作流已经加了 LoRA 节点
   - LoRA 会影响 I2V 生成的风格和细节
   - 问题是节点架构错误导致没效果

- 第一集已完成 → 所有镜头有成品图
- 用这些图做 I2V + LoRA → 生成第二集/扩展镜头
- T2V + LoRA 适合没有参考图时从零创作，一致性不如 I2V

### 结论
I2V + 参考图 + LoRA 是主角一致性最高的方案，当前路线正确，问题是工作流节点用错。
